AR Game

Racism

I made a simple Augmented Reality videogame with Unity and Visual Studio using C#.

Child labor

For this videogame I designed a maze with probuilder in Unity, I then used the Navmesh tool in order to be able to move the blue sphere automatically towards the green gem at the end of the maze whenever it was clicked with the mouse. Finally, I used Vuforia to put the whole game with the Navmesh into an image target so that it is playable in AR.

Boobs

The sphere is independent from the maze so the game can be played both in the intended way and also by moving the image target to which it is attached.
